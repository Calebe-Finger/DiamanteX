# Diamante X

## Introdução

Um programa de console que imprime um diamante (losango), de acordo com um número informado pelo usuário.

## Funcionalidades

- Possui tratamento para apenas aceitar números ímpares e inteiros.
- Imprime o losango, deixando o devido espaçamento para formar o desenho.
- Executa em loop, sem a necessidade de abrir o programa varías vezes.
> A execução é interrompida assim que o usuário digitar "S". <br>
Possui tratamento para aceitar tanto S maiúsculo quanto s minúsculo.
